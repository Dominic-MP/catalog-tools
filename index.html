This dashboard lists the current metrics tools for National Archives catalog data.


This project is maintained by Dominic-MP. Hosted on GitHub Pages — Theme by orderedlist

    Statalog

      This tool lists current general statistics for the catalog, to provide a snapshot of its scale and makeup. It displays counts for descriptions, authorities, objects, and web pages, with each broken down into further categories.

    Countributions

    (TBD)
      This tool lists current public contribution statistics for the catalog, to measure progress and engagement. It displays counts and other statistics for transcriptions, tags, and comments.

    Accounter

      This tool lists user data for registered catalog accounts, including overall and current-month counts. It displays both public data—like usernames, account types, and creation dates—as well as confidential information, such as email addresses, for authorized NARA account administrators.
